Employee Leave Management System
A bilingual (English/Amharic) web application for managing employee leave requests with Ethiopian calendar support.

🔗 Live Demo: https://lms-frontend-bkom.onrender.com

✨ Features
Authentication – JWT-based login with role-based access

Leave Management – Submit, approve, reject, and track leave requests

Bilingual Support – Full English/Amharic interface with language switcher

Ethiopian Calendar – Dates displayed in Ethiopian calendar for Amharic users

Leave Balance – Automatic tracking and deduction upon approval

Notifications – Real-time alerts for request status changes

Admin Controls – Manage leave types, view all requests, add comments

Audit Logs – Track all user actions for security

🛠️ Tech Stack
Backend: Node.js, Express, Sequelize ORM, MySQL, JWT
Frontend: React, Vite, i18next, Axios
